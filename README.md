# Hydrologic Data Processing
[![Open in MATLAB Online](https://www.mathworks.com/images/responsive/global/open-in-matlab-online.svg)](https://matlab.mathworks.com/open/github/v1?repo=drLKeen/HydroglogyVanderbilt)

This repository contains demonstration materials related to **hydrology**, developed for use in an academic setting at the **US Geological Survey** and **Vanderbilt University**. The content is intended to support teaching, learning, and exploration of hydrologic concepts using computational workflows.

## Repository Purpose
- Support hydrology instruction and demonstrations
- Provide example scripts, notebooks, or datasets for hydrologic analysis

## What You’ll Find Here
- Example code for retrieving hydrological data via USGS Water Data APIs
- Synthetic data created to demonstrate statistical analysis concepts
- pdf of presented slides with notes, links, and resources
